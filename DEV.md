## Stack

- Next.js 14 (App Router)
- Vanilla CSS + CSS Modules
- Framer Motion

## Run it

```bash
npm install
npm run dev
```

## Content

- Projects: `src/data/projects.ts`
- Writing: `src/data/writing.ts`
- Case studies: `src/app/work/[slug]/page.tsx`
- Images: `public/images/`

## Structure (high level)

```
src/
  app/            # routes + layouts
  components/     # UI pieces
  data/           # content data
  styles/         # global styles + tokens
public/           # assets
```

The design system principles in `.cursor/rules/` so I can stay consistent.
